# Feature: Quiz Management

## Overview
Allows users to create, edit, and publish quizzes with multiple question types and approval workflow.

## Details
- Supported question types: MCQ, fill-in-the-blank, true/false.
- Quizzes can be free or paid; paid quizzes require approval charge.
- Approval by Admin/Superadmin required before publishing.
- Quiz type (free/paid) is locked after approval.
- Only author, Admin, or Superadmin can modify quizzes before approval.
- Suggestions: Add quiz versioning/draft mode, question pools, and randomization for advanced delivery.
