# Feature: User Roles

## Overview
Defines three distinct user roles with specific permissions and responsibilities:

- **Superadmin**: Full platform control, including user management, content approval, system fee settings, and analytics access.
- **Admin**: Content moderation, quiz/course approval, user management, and analytics access.
- **User**: Can create quizzes/courses, purchase content, participate in leaderboards, and manage their wallet.

## Details
- Role-based access enforced throughout the platform.
- Only Admin/Superadmin can approve content and manage users.
- Users have limited access to their own content and wallet.
- Suggestions: Add user profile customization and activity logs for all roles.
